고급 설정 — 외부 시스템 직접 연동 (curl)

tmrw_web 외 외부 시스템(자체 랜딩, 디비카트 마이그레이션 등) 에서 직접 호출하려면 아래 명령어 참고.

사용법 (예시)

curl -X POST {{ request.scheme }}://{{ request.get_host }}/api/v1/leads ^
  -H "Authorization: Bearer <토큰>" ^
  -H "Content-Type: application/json" ^
  -d "{\"name\":\"홍길동\",\"phone\":\"01012345678\"}"

긴 payload·한글 깨짐 방지는 -d @payload.json 방식 추천.

curl -X POST {{ request.scheme }}://{{ request.get_host }}/api/v1/leads \
  -H "Authorization: Bearer <토큰>" \
  -H "Content-Type: application/json" \
  -d '{"name":"홍길동","phone":"01012345678","fields":{"업종":"도소매"}}'
